# 🌟 个人导航首页
一款纯前端实现的高颜值浏览器导航首页，集成实时时钟、天气查询、快捷搜索、自定义导航等功能，搭配毛玻璃视觉效果与流畅交互体验，开箱即用且高度可定制。

## 📸 效果预览
| 状态 | 效果描述 |
|------|----------|
| 默认状态 | 顶部实时时钟 + 居中搜索框 + 自定义导航按钮 + 底部一言/页脚 |
| 交互状态 | 点击时钟：时钟居中放大，背景模糊，显示日期/天气信息；点击搜索框：聚焦搜索，隐藏其他元素 |
| 视觉风格 | 毛玻璃磨砂质感 + 自定义艺术字体 + 平滑过渡动画 + 悬停发光效果 |

## ✨ 核心功能
- 🕒 **实时时钟**：秒级刷新，点击时钟居中放大并显示完整日期/节日
- ☁️ **天气查询**：基于IP自动定位，实时获取所在城市天气/温度
- 🔍 **快捷搜索**：自定义搜索引擎，搜索框打字机特效提示文字
- 🧭 **导航按钮**：可配置的快捷导航按钮，悬停放大+发光效果
- ✨ **视觉特效**：毛玻璃背景、平滑过渡、文字发光、点击交互
- 📜 **一言语录**：加载随机文案，支持点击刷新
- 🎨 **高度定制**：背景图、字体、颜色、导航按钮等均可自定义
- 🚀 **性能优化**：纯静态页面，无依赖，加载速度快

## 🎨 自定义配置（核心）
所有自定义项均在 `js/config.js` 中修改，无需改动核心代码：

### 基础配置
```javascript
const SITE_CONFIG = {
    title: "起始页",                // 网站标题
    favicon: "img/favicon.ico",    // 网站图标（本地/完整URL）
    backgroundImage: "https://t.alcy.cc/ycy", // 背景图URL
    hefengWeatherAPIHost: "你的和风天气APIHOST", // 天气接口API地址（申请后替换）
    hefengWeatherKey: "你的和风天气KEY", // 天气接口KEY（需自行申请）
    searchEngine: "https://www.baidu.com/s?wd=", // 搜索引擎地址
    placeholderText: "输入点什么吧...", // 搜索框提示文字
    // 更多配置...
};
```

### 导航按钮配置
```javascript
navButtons: [
        { text: "链接1", url: "https://xxx.com/" },
        { text: "链接2", url: "https://yyy.com/" },
        { text: "链接3", url: "https://zzz.com/" },
    // 新增按钮：{ text: "按钮文字", url: "跳转链接" },
],
```

### 页脚配置
```javascript
footer: {
    text: "©2026 ", // 页脚主文本
    links: [ // 页脚链接（支持多个）
        { text: "链接1", url: "https://xxx.com/" },
        { text: "链接2", url: "https://yyy.com/" },
    ],
    fontSize: "10px", // 字体大小
    color: "rgb(0, 0, 0)", // 文字颜色
    linkColor: "rgb(0, 0, 0)" // 链接颜色
}
```

### 其他自定义项
| 自定义内容 | 修改位置 |
|------------|----------|
| 背景图 | `config.js` → `backgroundImage` |
| 字体 | 替换 `ttf` 文件夹字体 + 修改 `main.css` 中 `@font-face` 路径 |
| 颜色/透明度 | `main.css` 中对应元素的 `color`/`background`/`rgba` 值 |
| 动画效果 | `main.css` 中 `transition`（过渡）/`transform`（变换）属性 |
| 搜索框/按钮样式 | `main.css` 中 `.search_input`/`.button_daohang` 相关样式 |

## 🛠️ 接口说明（天气功能）
1. **IP定位**：使用 `https://api.ip.sb/geoip/` 获取IP及地理位置（无需KEY）
2. **天气数据**：使用和风天气API（需自行申请免费KEY）
   - 申请地址：https://dev.qweather.com/
   - 替换 `config.js` 中 `hefengWeatherKey`以及`hefengWeatherAPIHost` 即可使用

## 🎯 交互说明
- **点击时钟**：时钟居中放大，显示日期/天气，背景模糊，隐藏搜索/导航/页脚
- **点击搜索框**：聚焦搜索，隐藏时钟/导航/页脚，背景模糊
- **点击页面空白处**：恢复默认状态
- **导航按钮**：悬停时放大+发光，点击跳转对应链接
- **搜索框**：回车触发搜索，支持placeholder打字机特效
- **底部一言**：点击可刷新文案
